Stellar Security Standards

We use the highest encryption standards to protect your data in transit, ensuring privacy and integrity across the galaxy.

1. Symmetric Encryption

ChaCha20: 256-bit key, optimal for mobile devices, resistant to timing attacks.

2. Message Authentication

Poly1305: MAC used with ChaCha20. Ensures data integrity and authenticity.

3. Key Exchange

Curve25519: X25519 elliptic curve. Fast and secure key establishment.

4. Hash Function

BLAKE2s: Modern fast hash, used for KDF and handshake authentication.

5. Key Production

HKDF: Based on BLAKE2s. Provides Forward Secrecy and session key rotation.
